# 📌 Sobre o Projeto

Este projeto foi desenvolvido como atividade prática do curso de Java do programa Capacita i-rede.

O objetivo foi aplicar os principais conceitos introdutórios da linguagem Java, trabalhando lógica de programação, estruturas condicionais, laços de repetição, arrays, manipulação de strings e organização de código em múltiplas classes.

---

# 📂 Estrutura do Projeto

- 1 Classe Main "Atividade1" responsável pela execução do menu principal
- 10 Classes separadas contendo cada atividade
- Organização modular para facilitar leitura e manutenção
- Cada classe representa uma questão da atividade proposta

---

# 🛠️ Tecnologias Utilizadas

- Java
- NetBeans IDE
- Programação Orientada a Objetos (POO)
- Estruturas de decisão e repetição
- Manipulação de Strings e Arrays

---

# 📋 Atividades Desenvolvidas

### Atividade 1
Verificação de número entre 10 e 20 utilizando operadores lógicos.

### Atividade 2
Exibição dos tipos de dados primitivos do Java com valores atribuídos.

### Atividade 3
Menu utilizando `switch` para soma, subtração e saída.

### Atividade 4
Geração da tabuada de um número utilizando laço `for`.

### Atividade 5
Substituição de letras em uma string.

### Atividade 6
Busca de nomes dentro de um array.

### Atividade 7
Cálculo de fatorial utilizando laço `for`.

### Atividade 8
Menu repetitivo utilizando `while`.

### Atividade 9
Separação do primeiro e último nome de uma pessoa.

### Atividade 10
Contagem de palavras em uma frase.

---

# 🎯 Objetivo de Aprendizagem

Praticar os fundamentos da linguagem Java através de exercícios focados em lógica de programação e estruturação de código.

---

# 🚀 Como Executar o Projeto

1. Baixe ou clone o projeto
2. Abra o projeto em uma IDE Java (NetBeans, IntelliJ ou Eclipse)
3. Execute a classe `Main`
4. Escolha a atividade desejada através do menu
