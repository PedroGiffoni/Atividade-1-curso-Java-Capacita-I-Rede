#README - Projeto Java | Unidade 1 - Tarefa 3
Este projeto foi desenvolvido como atividade prática do curso de Java do programa Capacita i-rede. O objetivo da tarefa foi aplicar conceitos fundamentais da linguagem Java, incluindo operadores lógicos, estruturas condicionais, laços de repetição, arrays, manipulação de strings e organização de classes.
#📚 Estrutura do Projeto
•	1 Classe Main responsável pela execução do menu principal
•	10 Classes separadas contendo cada atividade proposta
•	Cada classe representa uma questão da tarefa
•	Organização modular para facilitar manutenção e aprendizado
#🛠️ Tecnologias Utilizadas
•	Java
•	NetBeans IDE
•	Programação Orientada a Objetos (POO)
•	Estruturas de repetição e decisão
•	Manipulação de Strings e Arrays
#📋 Atividades Desenvolvidas
1. Verificação de número entre 10 e 20 utilizando operadores lógicos
2. Exibição de tipos primitivos do Java e seus valores
3. Menu utilizando switch para soma, subtração e saída
4. Tabuada de um número utilizando laço for
5. Substituição de letras em uma string
6. Busca de nomes dentro de um array
7. Cálculo de fatorial utilizando laço for
8. Menu repetitivo utilizando while
9. Separação de primeiro e último nome
10. Contagem de palavras em uma frase
#🎯 Objetivo do Projeto
Praticar os conceitos básicos da linguagem Java por meio de exercícios focados em lógica de programação e estruturação de código.
